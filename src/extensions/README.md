# FunkPHP Intellisense

Context-aware autocompletion and structural snippets optimized for the FunkPHP monolithic framework lifecycle.

## Features

- **Validation Studio Rules:** Intelligent inline rule generation when typing rules within localized validation directories.
- **SQL Schema Hints:** Fast table join keyword discovery boundaries inside query workspaces.
- **Template Engine Mapping:** Clean properties lookups for local frontend template file variables using `{{d.` structures and the like.
